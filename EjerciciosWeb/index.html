¿Cómo funciona?

Una página HTML tiene esta estructura básica:

  1. DOCTYPE: indica que el archivo es HTML.
  2. html: envoltura principal del documento.
  3. head: datos del documento como título y estilos.
  4. body: contenido visible de la página.
