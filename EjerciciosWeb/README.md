# Ejercicios Web

## Como usar esta carpeta
1. Abri un archivo `index.html` y presiona `Alt + B` (si tenes Live Server) o clic derecho -> Open with Default Browser.
2. Guarda cambios y refresca el navegador.
3. Practica un ejercicio por carpeta.

## Estructura
- 01-html-basico: etiquetas HTML
- 02-css-basico: estilos CSS
- 03-json-xml: practica de formatos
